# Python Game

## 담당 기능

#### 더게임오브데스 (game2.py)

1. 술래는 2 이상 (총 인원\*2) 이하의 숫자를 고릅니다.
2. 각자 누군가를 손가락으로 지목합니다.
3. 술래의 손가락부터 시작되어, 지목된 사람마다 숫자가 1부터 카운트됩니다.
4. 술래가 골랐던 숫자가 카운트되며 지목당한 사람이 패배하게 됩니다.
